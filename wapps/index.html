Utility Web Apps from BMSS

Here are some simple Browser based apps to try and use for FREE from BMSS repository.

Map

This is a simple mapping application that allows viewing GIS data in KMZ/KML and GeoJson format interactively. Key Features include...

Viewing :

Display KMZ/KML and GeoJson vector layers with attributes in popups on click. Allows multiple layers of data, i.e., you can upload and view multiple data layers at the same time.
You may configure layer wide various display settings and save the entire setup as a project file to disk, which you can simply load as a project and view.

It provides various basemaps like Street and Satellite. Allows locating current user location. View full screen and extend map view to loaded data for better viewing experience.

Search and Filter:

The app offers search at different levels. You may search place names from the Nomanitim database or based on the layers being loaded locally. It filters a list, matching all entities by the text typed by the user. Clicking on any of the searched item would make the map pan and zoom into the appropriate object.

Routing & Navigation:

The app also provides for a simple routing and navigation interface. It allows right clicking on the map to set starting and ending points to calculate the shortest routes. It displays alternative route if any and distincts with a different color. You can click on any of the route to view the detailed navigation chart on the panel.

The navigation panel allows selection of speed levels to view estimated time of travel [ETA]

TabLink

This app is for managing attribute columns in the GeoJson file which you created using Google My Maps and MyGeoData. Initially this GeoJson does not have any attribute columns. Without attribute columns you cannot attach information to the individual GeoJson geometries. And without information you have nothing to display in the popups in MapViewer. TabLink handles it easily. You just load the GeoJson file and Geotab displays the columns and the inherent code of the GeoJson file in a simple and easy interface. 

You can do the following changes to the uploaded GeoJson file and save/download the updated version in your default download folder.

1. Add new column.
2. Delete an existing column.
3. Change column order.
4. Rename a column header

Its small and simple.

TabEdit

This is the application you shall be using most often to manage your data to be displayed in Map. Once you have added the attribute table columns to your GeoJson file, or any GeoJson file with attribute columns, you can simply load this file here and add, edit or remove data.

It also provides a visual presentation of the loaded geojson file in a map for visual identification and selection, making it even easier for the user. Here is a screenshot of the Guwahati Municipality Ward geojson file loaded into TabEdit..

It allows real time search and filter (search as you type) on all fields of the geojson file. Resizing of display fonts and columns by dragging. It also allows hiding and viewing columns. There is an option to view the geojson geometry in a map with basemap, which allows map panning and zooming. The map is also interactive in a way, that it automatically locates the appropriate data row on click.

Attribute Editor not only allows saving the updated geojson file to your choice of location in local disk, but it also allows downloading the attribute data in Excel.


General Instructions

Installation : None required. All apps are hosted in our server. All you have to do is open the url's in your browser. We recommend Chrome & Edge. Safari might run okay but there are some real issues with Firefox. These apps run on local servers too. Map would require one that can handle PHP, otherwise lightweight Baby Web Server would also do. We recommend USBWebServer which can handle PHP and Apache.

Hardware Prerequisites : Nothing very high end. Standard configurations would do.
